# Virtual assistant using Speech recognition

## Tasks it can perform
* Wish you according to time
* Search Wikipedia
* Open youtube
* Tells time
* open notepad
* opens paint
* Takes screenshot
* Open powershell
* Open Calculator
* Open Task Manager
* Tell Jokes
* Search Google
* Lock Window
* Shutdown Computer
* Find Location on GMAP
* Restart Computer
* Hibernate Computer
* Log Out Computer
* Answers some questions eg. Who am I ?

## How to Setup?
```bash
cd /path/to/dir
python -m pip install -r requirements.txt
```
## How to Run ?
```bash
python main.py
```
